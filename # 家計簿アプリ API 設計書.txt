# 家計簿アプリ API 設計書

## 🗂️ リソース一覧

- `transactions`：入出金履歴
- `summary`：月別の入出金集計
- `categories`：カテゴリ一覧（オプション）

---

## 🔗 エンドポイント一覧

| メソッド | エンドポイント              | 説明                         |
|----------|-----------------------------|------------------------------|
| GET      | `/api/transactions`         | 入出金履歴一覧を取得         |
| GET      | `/api/transactions/:id`     | 特定の履歴詳細を取得         |
| GET      | `/api/summary/:year/:month` | 指定月の入出金集計を取得     |
| GET      | `/api/categories`           | カテゴリ一覧を取得（任意）   |

---

## 📥 HTTP メソッド別設計

### ✅ GET `/api/transactions`

- **説明**：すべての入出金履歴を取得
- **レスポンス例**：
```json
[
  {
    "id": 1,
    "date": "2025-10-01",
    "type": "income",
    "amount": 50000,
    "category": "給料",
    "description": "10月分の給料"
  },
  {
    "id": 2,
    "date": "2025-10-05",
    "type": "expense",
    "amount": 12000,
    "category": "食費",
    "description": "スーパーで買い物"
  }
]
